Lab 與互動實作 ​

原課程有七個 Lab，會提供 live access 到 IBM Z server。本站不重建 Lab runtime，也不保存任何登入狀態、命令結果或學習者進度；這裡只整理學習順序、目的與原課程入口。

原課程中的 Lab 與互動活動需要在 IBM Learn 環境中完成。本站保留活動定位、學習目的與原課程連結，協助你安排學習順序。

7項原課程實作活動
z/OS 命令與面板入門Lab

練習 1：存取 Lab 環境

Exercise 1: Access your lab environment

透過 IBM Remote Lab Platform 進入原課程 Lab session，確認課程套件、IBM ID 登入與主機環境入口。

學習目的

  • 確認 IRLP / Skytap Lab 環境可用
  • 取得後續練習會使用的系統與 TSO credential

建議先完成

  • TSO 與 ISPF 入門
  • Lab：熟悉系統
  • 開始 Lab 前的檢查清單

此活動需要原 IBM Learn 環境與授權 session。本站保留學習順序與目的，實作請回原課程完成。

開啟原課程活動前往 IBM Learn 課程
z/OS 命令與面板入門Lab

練習 2：配置新的資料集

Exercise 2: Allocate new data sets

練習建立資料集，理解資料集命名、配置屬性與後續編輯作業的關係。

學習目的

  • 熟悉資料集配置流程
  • 建立後續資料集操作練習的材料

建議先完成

  • 熟悉系統
  • TSO 與 ISPF 入門

請在原課程環境中完成，避免在本站保存任何學習者帳號、環境或結果資料。

開啟原課程活動前往 IBM Learn 課程
處理資料集Lab

練習 3：ISPF 編輯器主要命令

Exercise 3: ISPF editor primary commands

使用 ISPF 編輯器的主要命令處理資料集內容。

學習目的

  • 辨識主要命令輸入位置
  • 練習儲存、取消與瀏覽常用編輯動作

建議先完成

  • 配置資料集
  • 使用資料集

此 Lab 以原課程環境中的資料集為準；本站只提供學習定位。

開啟原課程活動前往 IBM Learn 課程
處理資料集Lab

練習 4：ISPF 編輯器行命令

Exercise 4: ISPF editor line commands

透過行命令操作特定資料列，建立對 ISPF 編輯器操作模式的實務理解。

學習目的

  • 練習在行命令區輸入操作
  • 區分主要命令與行命令的用途

建議先完成

  • ISPF 編輯器主要命令
  • 使用資料集

請回 IBM Learn 原活動執行，並依課程環境中的資料集名稱操作。

開啟原課程活動前往 IBM Learn 課程
處理資料集Lab

練習 5：複製、移動、重新命名、刪除資料集與成員

Exercise 5: Copy, move, rename, delete data sets and members

練習資料集與成員的常見管理動作，包含複製、移動、重新命名與刪除。

學習目的

  • 理解資料集與成員管理的日常工作流
  • 建立操作前確認目標名稱的習慣

建議先完成

  • 資料集配置
  • ISPF 編輯器行命令

此活動涉及原環境中的資料集狀態，請在 IBM Learn 中依步驟完成。

開啟原課程活動前往 IBM Learn 課程
處理資料集Lab

練習 6：資料集清單

Exercise 6: Data set lists

透過資料集清單尋找、篩選與進入資料集，讓前面建立與管理的資料集可以被有效使用。

學習目的

  • 熟悉資料集清單瀏覽
  • 練習用命名模式定位資料集

建議先完成

  • 複製、移動、重新命名、刪除資料集與成員

資料集名稱與可見清單依原課程環境而定，請回原課程執行。

開啟原課程活動前往 IBM Learn 課程

Exercise 7: Using TSO Commands

以 TSO/E 命令完成常見操作，將前面透過 ISPF 面板建立的理解延伸到命令列。

  • 熟悉 TSO/E 命令輸入與結果判讀
  • 比較面板操作與命令操作的差異

建議先完成

  • TSO 命令 - 第 1 部分
  • TSO 命令 - 第 2 部分

請在 IBM Learn 原環境中完成，本站不重建主機連線或保存命令執行結果。
