# 互動練習

本頁將原課程 H5P short quiz 轉成靜態學習練習。這裡不計分、不保存作答紀錄；badge quiz、certificate、course survey 與學習者 attempt 記錄仍留在 IBM Learn。

<PracticeQuestions />

## 使用原則

- 題目只作學習確認，不替代 IBM Learn 正式評量。
- 不公開 pass/fail、badge、certificate 或 attempt 語氣。
- 每題需要來源、答案、解析與回到課程複習的提示。

正式 badge quiz、certificate、course survey 與 practitioner badge claim 請回 [IBM Learn 原課程](https://learn.ibm.com/course/view.php?id=7419) 完成。
