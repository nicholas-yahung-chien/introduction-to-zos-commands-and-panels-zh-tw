# 互動練習

本頁將原課程可作為學習檢核的 quiz 轉成靜態練習。重新盤點與授權確認後，靜態練習共收錄 44 題：Short quiz 19 題，加上 Badge Quiz 題庫 25 題。

這裡不計分、不保存作答紀錄；正式 Badge Quiz attempt、certificate、course survey 與 badge claim 仍需回 IBM Learn 完成。

## 題庫來源

- Short quiz：6 個來源，共 19 題，包含 Exercises 1-2 的 Moodle quiz 8 題與 Exercises 3-7 的 H5P short quiz 11 題。
- Badge Quiz 題庫：每次正式 attempt 顯示 20 題；目前從 2 次 completed review 合併去重後收錄 25 題。
- Badge Quiz 題目已依 IBM 團隊主管授權重製為靜態練習；正式評量成績、attempt 紀錄與 badge 仍以 [IBM Learn 原課程](https://learn.ibm.com/course/view.php?id=7419) 為準。
