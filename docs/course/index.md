## 建議學習方式

1. 先閱讀「z/OS 命令與面板入門」，熟悉 TSO/E、ISPF 與 Lab 環境。
2. 依序完成「處理資料集」與「TSO 命令」兩個單元，建立資料集與命令操作手感。
3. 每支影片先看完一次，再回到影片下方摘要確認學習重點與重要術語。
4. 遇到不熟悉的 z/OS 名詞時，搭配詞彙表查閱原文術語與繁體中文說明。
5. 完成單元後到「互動練習」確認概念，答錯時回到對應影片或 Lab 複習。
6. 需要實作時，閱讀「Lab 與互動實作」後回到 IBM Learn 原課程完成。

## 學完後你會理解

- TSO/E 與 ISPF 如何作為 z/OS 互動式操作入口。
- Sequential data set、PDS、PDSE、member 與 qualifier 等資料集概念。
- 如何使用 ISPF editor、dataset list 與常見 TSO commands 完成基本操作。
- Lab 練習與 Badge Quiz 題目涵蓋的主要操作重點。
