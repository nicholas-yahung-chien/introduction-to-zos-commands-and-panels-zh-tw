# 授權資訊

本教材整理自 IBM Learn 課程 `Introduction to z/OS Commands and Panels on IBM Z`，作為台灣繁體中文化學習版本。

- 課程來源：IBM Learn `Introduction to z/OS Commands and Panels on IBM Z`
- 課程網址：https://learn.ibm.com/course/view.php?id=7419
- 影片搭配 `zh-Hant-TW` WebVTT 字幕提供
- 詞彙表與教材用語以 IBM Z、z/OS、TSO/E、ISPF 與資料集相關術語為基礎整理
- Badge Quiz 題目已依 IBM 團隊主管授權重製為非計分靜態練習；正式 attempt、certificate、badge claim 與學習者進度仍以 IBM Learn 為準

本網站保留原始 IBM Learn 活動來源連結，方便學習者回到原課程脈絡查閱。授權與使用範圍請依 IBM 課程授權與內部審核結果為準。
