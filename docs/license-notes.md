# 授權資訊

本網站是 `Introduction to z/OS Commands and Panels on IBM Z` 的台灣繁體中文靜態學習網站骨架，原課程位於 [IBM Learn](https://learn.ibm.com/course/view.php?id=7419)。

## 來源與使用範圍

- 原課程、影片、Lab、評量與 badge 內容屬 IBM 或其授權平台所有。
- 本站整理課程順序、學習目的、術語、原課程連結與已授權重製的靜態練習題，供學習導覽與自我檢核使用。
- Badge Quiz 題目已依 IBM 團隊主管授權重製為非計分靜態練習；正式 Badge Quiz attempt、certificate、badge claim、attempt 與學習者進度仍需回 IBM Learn 完成。
- 影片與字幕需在取得授權後才能部署於本站或媒體 host。

## 公開資料佐證

IBM Learn 課程頁需登入才能檢視完整內容；本專案建立時也參考公開可見的課程目錄描述，例如 Pluralsight 的 `Introduction to z/OS Commands and Panels` 課程頁與其他公開課程目錄摘要，用於確認課程主題、影片名稱、Lab 數量與學習目標。

## 維護注意

- 不在公開頁放置登入 session、attempt id、Lab runtime 或學習者資料。
- 正式測驗題目只有在取得明確授權後才可重製；本版 Badge Quiz 題庫已依授權納入靜態練習。
- 若未來加入影片，需同步提供合法媒體來源、HLS、英文字幕、繁體中文字幕與字幕檢查紀錄。
