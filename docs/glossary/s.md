# IBM Z 詞彙表：S

## Search-For (ISPF)

ISPF Utilities option `3.14` 與 `3.15` 的字串搜尋工具，可在指定 data set 或 members 中搜尋文字。

## SEND command

TSO 中傳送訊息給其他使用者的命令。課程題目示範以 `SEND` 搭配 `USER(...)` 指定收件使用者。

## Sequential Data Set

記錄以順序方式連續儲存的資料集，通常不包含 members。配置 sequential data set 時，directory blocks 通常為 0。

## SuperC (ISPF)

ISPF Utilities option `3.12` 與 `3.13` 的資料比較工具，用於比較 data set、member 或文字內容差異。
