## Catalog

z/OS 用來記錄資料集名稱與其所在磁碟區等資訊的系統資料結構。透過 catalog，使用者可以用資料集名稱存取資料，而不必直接指定實體位置。

## Catalog alias

Catalog 中的別名項目，通常用來將特定 high-level qualifier 對應到指定的 user catalog。

## CHANGE command

ISPF editor 中用來取代文字的 primary command。可指定要尋找的字串、替換字串與搜尋範圍。

## CLIST

Command list 的縮寫，是 TSO/E 中可執行的一組命令序列。CLIST 可用來自動化重複性的 TSO 操作。

## Command line

使用者輸入命令的位置。在 ISPF panel 中通常顯示為 Command ===> 或 Option ===> 欄位。

## Command parameter

定義正在建立的命令的參數。

## COMPRESS command

TSO/E 中用於整理 PDS 空間的命令。COMPRESS 可回收刪除或更新 member 後留下的可用空間。

## Cylinder

CKD 裝置上具有固定磁軌數的儲存單元。
