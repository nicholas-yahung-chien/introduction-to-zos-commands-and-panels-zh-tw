# IBM Z 詞彙表：F

[返回詞彙表索引](./)

## FIND command

ISPF editor 中搜尋字串的命令。找到目標後，可用 RFIND 或對應 function key 重複搜尋。

## FREE command

TSO 中用來釋放不再需要的資料集配置。它和 allocation 是一組相對概念：先 allocate 使用資源，用完後可 free。

## Function key (ISPF)

ISPF panel 下方顯示的 PF key 功能提示，例如 `F1=Help`、`F3=Exit`、`F5=RFIND`、`F7=Backward`、`F8=Forward`。不同 panel 的 function key 可能有不同定義。
