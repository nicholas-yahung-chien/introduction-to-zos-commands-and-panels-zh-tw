z/OS 命令與面板詞彙表 ​

本詞彙表整理這門課的核心術語。IBM 產品名、介面名與常見縮寫保留英文，中文說明採台灣繁體用語。

Data set ​

資料集。z/OS 中用來存放資料的主要單位，可依組織方式與用途分成不同型態。

ISPF ​

Interactive System Productivity Facility。z/OS 上常用的面板式互動環境，可用於瀏覽、編輯與管理資料集。

ISPF editor ​

ISPF 編輯器。透過主要命令與行命令編輯資料集或成員內容。

Line command ​

行命令。在 ISPF 編輯器或清單畫面的行命令區輸入，針對特定資料列執行操作。

Member ​

成員。Partitioned Data Set 中的個別項目，常用來存放程式、JCL 或設定內容。

PDS ​

Partitioned Data Set。分割式資料集，可包含多個 member。

Primary command ​

主要命令。在 ISPF 畫面的命令列輸入，通常作用於目前面板或編輯工作階段。

TSO/E ​

Time Sharing Option/Extensions。z/OS 上的互動式命令環境，讓使用者輸入命令並取得回應。
