On this page

TSO 命令 ​

本單元把前面透過 ISPF 面板建立的操作經驗延伸到 TSO/E 命令。學習者會練習命令語法、結果判讀與最後一個 hands-on Lab。

影片學習摘要 ​

TSO 命令 - 第 1 部分 ​

這段學習開始介紹 TSO/E 命令的用途，讓學習者從面板操作延伸到命令輸入。

學習重點

  • TSO/E 命令可直接執行互動式工作
  • 命令語法與參數順序需要精準
  • 可與資料集操作相互搭配

重要術語

  • TSO/E command
  • Command syntax
  • Dataset

TSO 命令 - 第 2 部分 ​

這段學習延續 TSO/E 命令，強化常用命令與命令結果判讀。

學習重點

  • 練習辨識命令回應與錯誤訊息
  • 建立命令操作前檢查參數的習慣
  • 理解何時使用命令、何時回到 ISPF 面板

重要術語

  • Command response
  • Return code
  • ISPF

Lab：使用 TSO 命令 ​

這段學習說明第七個 Lab 的操作目標，協助學習者先理解要完成的 TSO/E 命令任務。

學習重點

  • 先回顧 TSO/E 命令語法
  • 確認要使用的資料集與命令目標
  • 完成後可用結果檢查操作是否符合預期

重要術語

  • TSO/E
  • Lab
  • Command

原課程活動 ​

順序活動類型建議用途
1TSO Commands - part 1影片認識 TSO/E 命令用途與基本語法。
2TSO Commands - part 2影片延伸常用命令與結果判讀。
3Lab - Using TSO commands影片確認最後一個 Lab 的操作目標。
4TSO/E Command Reference File檔案作為命令查詢參考。
5Exercise 7: Using TSO commandsLab在原環境中執行 TSO/E 命令。
6Short quiz: Exercise 7互動內容在 IBM Learn 檢查本單元理解。
7Discussion forum討論區回原課程參與課程討論。

學習提醒 ​

  • TSO/E 命令的大小寫、標點、括號與參數順序都可能影響結果。
  • 命令操作前先確認資料集名稱與目標，操作後再閱讀回應訊息。
  • 若對命令語法不確定，回到影片與 Lab 指示逐步比對。
